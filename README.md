# Number-Plate-Recognition
📌 Number Plate Recognition System
🔹 Project Overview

This project is an Automatic Number Plate Recognition (ANPR) system that detects vehicle number plates from images or video streams and extracts the text using computer vision and OCR techniques.

🔹 Features

🚗 Detects vehicle number plates from images/videos

🔍 Uses image processing techniques for plate localization

🔤 Extracts text using OCR (Optical Character Recognition)

⚡ Real-time detection (if using webcam/video)

📊 High accuracy with preprocessing


🔹 Technologies Used

Python 🐍

OpenCV

NumPy

Tesseract OCR
